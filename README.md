# NOTA BENE

NOTA BENE è un'applicazione web realizzata come progetto del corso di Ingegneria del Software (a.a. 2024/2025) presso l'Università di Bologna.

## Descrizione
L'applicazione consente agli utenti di creare, organizzare e condividere note testuali (max 280 caratteri). Il sistema supporta il versionamento delle modifiche, la collaborazione e la gestione dei permessi.

## Stack tecnologico
- Java
- GWT (Google Web Toolkit)
- Maven (build system)
- MapDB (persistenza dei dati)
- Gson (interazione JSON)
- JUnit (unit testing)

## Struttura del progetto
Il progetto segue le best practice Agile con metodologia Scrum. Le funzionalità sono gestite tramite GitHub Project e sviluppate su feature branch.

## Setup rapido
1. Clonare la repository  
2. Eseguire `mvn clean install`  
3. Eseguire l’applicazione tramite Jetty (plugin Maven)  
4. Aprire il browser su `http://localhost:8080`
